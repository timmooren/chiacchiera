# Language Learning Chat

A web app for practicing Italian or Spanish through casual conversation with an AI language partner. Pick a language and a topic, chat away, and get gentle corrections of your messages as you go.

## Setup

1. Install dependencies:

   ```sh
   npm install
   ```

2. Copy `.env.example` to `.env` and add your Anthropic API key:

   ```sh
   cp .env.example .env
   ```

3. Start the server:

   ```sh
   npm start
   ```

4. Open http://localhost:3000 in your browser.
